# VirtualDeck - CardDeck iOS App

A SwiftUI card game application.

## Building

This project uses GitHub Actions to automatically build and deploy to Appetize.io.

### Setup

1. Push this repository to GitHub
2. Add your Appetize API token as a repository secret named `APPETIZE_API_TOKEN`
3. Push to `main` branch to trigger the build

### Getting Appetize API Token

1. Sign up at [appetize.io](https://appetize.io)
2. Go to your account settings
3. Generate an API token
4. Add it to GitHub repository secrets

## Local Development

This project requires Xcode 15+ on macOS to build locally.
